# MealTracker
MealTracker is an App where one can track their Meals
*The User can:
1. Add New Meal.
2. View all Meals
3. Edit Meal Details
4. Delete Meal.
5. Sort Meals based on their calories level
This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 1.5.0.
